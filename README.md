## my website

### 项目介绍

本项目是一个基于Next.js的个人网站，包含个人简介、技能展示、项目展示、博客文章、联系信息等模块。网站采用响应式设计，适配不同设备，具有良好的用户体验。

### 技术栈

- React：前端框架
- React Router：路由管理
- Zustand：状态管理
- Fetch：HTTP请求库
