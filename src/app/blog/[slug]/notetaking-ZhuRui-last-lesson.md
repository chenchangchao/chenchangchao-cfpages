>- 在我看来,死亡是⽣命的⼀部分,它并⾮对⽣命的否定,⽽是肯定,是重⽣｡它和永⽣相对,给了世界⼀个重⽣的机会,死亡代表的重⽣不⼀定是个⼈灵魂的重⽣,⽽是世界的万象履新｡如果这个世界的⼀切均为永⽣,那么新⽣物将永远不会出世,世界将没有空间,充满⽼旧,这是很可怕的｡

>- ⼀个⼈应该学会孤独,在⼼灵深处保守⼀⽚孤独的天地,默默耕耘⾃⼰的思想,冷静但热情地看待这个世界。不带任何幻想和猜想,同时⼜⾮常积极地去活｡

>- 如果现在您可以重返课堂,您想给学⽣哪些嘱托呢？
>- 第⼀,⼈⽣没有迈不过去的坎⼉,包括死亡｡第⼆,要时刻记住社会做贡献,不要为⼩我所困｡第三,要与⼈为善,不能踩着别⼈的利益获取⾃⼰的利益｡我告诉我带的硕⼠⽣和博⼠⽣,做学问就像苦海⾏⾈,需要强⼤的毅⼒,不适合所有⼈,甚⾄可以说对⼤部分⼈来说都不适合,但是⼀旦选择了这条道路,你就必须主动地阅读⼤量的各种各样的图书,你迟早能从这种艰⾟中体验到精神的满⾜和升华｡
>- 在我看来,“内卷”是欲望的博弈,“躺平”则是欲望的消磨,代表着⼀种低欲望甚⾄⽆欲望的⽣存状态｡这似乎是⼀个虚假的两难选择｡因为我们只要简单地了解欲望机制背后的⼀些道理,我们甚⾄可以设想出第三种情——⼀种⾼欲望､低内耗的和谐社会｡
>- 法国哲学家勒内•基拉尔设想过这样⼀个场景:⼩男孩⾛进他的卧室,尽管满屋⼦都是玩具,但是他不知道该选哪⼀个,⽆聊之中他随便拿起⼀辆玩具⻋｡在⼩男孩正要放下这辆⻋去拿另⼀个玩具时,他妹妹⾛了进来｡妹妹看⻅哥哥⼿中的⻋,就向他索要,但是哥哥不愿意给她,于是两个⼈发⽣⼝⻆,甚⾄起了冲突｡

>- 让我们来设想⼀下,所谓的“内卷”是不是属于这种模仿欲望机制下的情形｡我个⼈觉得,我们之所以“内卷”,并不⼀定是因为我们⼈多,也不⼀定是因为资源少,⽽恰恰是因为我们的欲望被外在的机制单⼀化,于是我们就像那对兄妹⼀样,对某⼀事物进⾏⽆谓的争⽃｡
>- 相反,如果我们知道⾃⼰想要什么,真正联结⾃⼰的欲望和事物的价值,也许我们就可以⾃然地实现欲望的多元化｡⽽当⼀个社会的欲望多元化之后,所谓的资源的稀缺也应该会相对缓解,⼈与⼈之间的冲突也会相应地缓解｡所以并不是说在内卷之外,只有低欲望或者⽆欲望,⽽是我们可以通过对欲望的培养､发展,以及对欲望机制的⾃主性改造,进⼊⼀个⾼欲望但低内耗的⼤同社会｡

>- 哲学家的最后⼀课是朱锐⽼师的哲学绝唱｡读这本书的时候,我仿佛看到他讲课时的情景｡他在学⽣⾯前微笑着,没有悲伤和对死亡的畏惧｡在对他深表感佩的同时,我深感思想能够通达⽣命的本质｡这本书源⾃朱锐⽣命深处的豁达,深切表明哲学家对这个世界的深爱和勇⽓,表明对⾃我和世界的信念,⽽未经审视的⽣活是不值得过的｡------臧峰宇,中国⼈⺠⼤学哲学院院⻓
